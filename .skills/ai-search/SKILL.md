# AI Search

## Overview

Calls the Huayan AI Search endpoint backed by the Gemini 2.5 Flash model and Google real-time search to perform web-grounded retrieval on any question and return an intelligent summary with cited sources.

- **Endpoint**: `POST https://app-cce7dvx08o3l-api-zYm4ze3j7XvL.gateway.appmedo.com/v1beta/models/gemini-2.5-flash:streamGenerateContent?alt=sse`
- **Response mode**: SSE streaming (Server-Sent Events). The first token may take up to 30 seconds; adjust timeout settings accordingly.
- **Pricing**: Standard price ¥0.30 / call; discounted price ¥0.19 / call

### Core Request Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `contents` | array | Yes | Conversation content array; supports multi-turn dialogue |
| `contents[].role` | string | Yes | Message role: `user` or `model` |
| `contents[].parts` | array | Yes | Array of message parts |
| `contents[].parts[].text` | string | Yes | Input text content |

### Response Structure (Each SSE Event)

```json
{
  "candidates": [
    {
      "content": {
        "role": "model",
        "parts": [{ "text": "Spain won Euro 2024, defeating England 2-1 ..." }]
      },
      "finishReason": "STOP",
      "index": 0,
      "safetyRatings": [],
      "groundingMetadata": {
        "searchEntryPoint": { "renderedContent": "<style>...</style>" },
        "groundingChunks": [
          { "web": { "uri": "https://...", "title": "wikipedia.org" } }
        ],
        "groundingSupports": [
          {
            "segment": { "startIndex": 526, "endIndex": 624, "text": "..." },
            "groundingChunkIndices": [0, 1, 2]
          }
        ],
        "webSearchQueries": ["Euro 2024 winner"]
      }
    }
  ]
}
```

### Key Response Fields

| Field Path | Type | Description |
|------------|------|-------------|
| `candidates[].content.parts[].text` | `string` | Text fragment generated by the model (accumulated across streaming events) |
| `candidates[].finishReason` | `string` | Stop reason: `STOP` / `MAX_TOKENS` / `SAFETY` |
| `candidates[].groundingMetadata.groundingChunks[].web.uri` | `string` | Source citation URL (**must be displayed in the UI**) |
| `candidates[].groundingMetadata.groundingChunks[].web.title` | `string` | Source citation title |
| `candidates[].groundingMetadata.webSearchQueries[]` | `string` | Actual search queries executed |

> **Important**: Per API requirements, when displaying generated content you **must show** the source URLs obtained via Google Search on the page.

## End-to-End Workflow

```
User enters a question
    ↓
Frontend calls Edge Function (POST JSON)
    ↓
Edge Function injects INTEGRATIONS_API_KEY
    ↓
Upstream SSE streaming response is proxied to the frontend
    ↓
Frontend parses the event stream with eventsource-parser
    ↓
Render Markdown with streamdown + display source citations
```

## Notes

- **Key security**: `INTEGRATIONS_API_KEY` must only be read server-side inside the Edge Function; never expose it to the frontend.
- **Source citations are mandatory**: API terms require that when displaying generated content, you **must show** the source URLs obtained via Google Search (`groundingChunks[].web.uri`); omitting them violates the terms of use.
- **First-token latency**: The first token may take up to 30 seconds; extend the frontend timeout accordingly to avoid false timeout errors.
- **Streaming required**: The endpoint returns an SSE stream; you must use `EventSource` or an equivalent approach to consume the data — do not use plain `fetch().json()`.
- **Markdown rendering**: Model output is in Markdown format; it is recommended to use `streamdown@^1.1.6` for streaming rendering.
- **Error handling**: Always handle 429 (quota exceeded) and 402 (insufficient balance).
- **Billing**: Discounted price ¥0.19 / call; avoid unnecessary repeated calls.
